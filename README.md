# telescope-vimwiki.nvim

Look for your vimwiki pages using telescope!

### Requirements:

- neovim (>= 0.5.0)
- vimwiki

### Installation

```require'telescope'.load_extension'vimwiki'```


### Usage

```:Telescope vimwiki```
